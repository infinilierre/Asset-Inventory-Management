# Asset & Inventory Management System 🚀

A full-stack project focused on managing organizational resources through a dynamic web interface and a structured relational database.

## 🔑 Key Features
- **Data Integration:** Seamless communication between PHP backend and SQL database.
- **Dynamic UI:** Responsive dashboard built with Vanilla JavaScript and Fetch API.
- **Asset Lifecycle:** Tracks item categories, quantities, and real-time status updates.

## 🛠 Technical Stack
- **Languages:** PHP, JavaScript (ES6+), SQL.
- **Markup/Styling:** HTML5, CSS3.
- **Infrastructure:** Tested on Linux-based environments.
- **Security: Mentioning:** Prepared Statements for SQL Injection prevention. 
- **Installation:** 1. Import the SQL file. 2. Configure db_config.php. 3. Run on local server. 

## 🎓 Project Context
Developed to demonstrate proficiency in core web technologies and database management, bridging academic theory with practical enterprise application.
